# 1880 — Rules Reminder

## Quick Reference
- **Players:** 3–7
- **Bank:** ¥37,860
- **Starting Cash:** ¥600 (3p) / ¥480 (4p) / ¥400 (5p) / ¥340 (6p) / ¥300 (7p)
- **Capitalization:** 50% on float, second 50% when 50% shares sold AND after first 3-train
- **Float:** 20% (A1) → 30% (B1) → 40% (B3) → 60% (C2) — changes by phase
🤖 - **Shares:** Variable president (20–40%), rest 10%. Excess shares removed if president > 20%
- **President cert size:** chosen at par. 40% = 1 phase building permit, 30% = 2 phases, 20% = 3 phases
🤖 - **Sell Movement:** down per 10% sold
🤖 - **Unsold Dividends:** market → corp
🤖 - **Pool Limit:** 80%
🤖 - **Sell Order:** sell/buy
- **Max Ownership:** 100% (unlimited)
- **No separate IPO/bank pool** — all shares in one bank
- **Currency:** ¥

## Specialties
- **Variable president cert (20–40%)** — president chooses size at par, determines building permit duration
- **Foreign Investors (A1–A7)** — operate like mini-corps, lease trains, always withhold, join with associated company when connected
- **Building permits (A/B/C/D)** — gate which corps can lay track per phase
- **Share Dealing Rounds (SDR)** — triggered by last train of a type sold/removed (not regular SR)
🤖 - **Communist takeover** at phase B3 — closes stock market
🤖 - **Stock exchange reopens** at phase C2
🤖 - **Token cost doubled** at phase D1
- **50% capitalization** — first half on float, second half later

## Stock Market Movement
🤖 - **Pay dividend:** right 1 (up if at edge)
🤖 - **Withhold:** left 1 (down if at edge)
🤖 - **Sell shares:** down 1 per 10% sold
🤖 - **Sold out:** up 1 at end of SDR (shares on Foreign Investors don't count)

## Auction
🤖 - Draft: each player picks one private in order

## Round System
- SDR (Share Dealing Round) → OR × phase-dependent
- SDRs triggered by last train of type sold/removed
- Phase A: 1 OR, Phase B–D: 2 ORs

## Not Implemented
- Variable president cert size (20–40%) at par
- Foreign Investors (A1–A7) — separate entities
- Share Dealing Rounds (triggered by train events, not regular SR)
- 50% capitalization (two-stage: float + later unlock)
- Building permits gating track laying
- Communist takeover / stock market closure
- Phase-dependent float percent changes
