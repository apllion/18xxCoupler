# 1880 — Rules Reminder

## Quick Reference
- **Players:** 3--7
- **Bank:** ¥37,860
- **Starting Cash:** ¥600 (3p) / ¥480 (4p) / ¥400 (5p) / ¥340 (6p) / ¥300 (7p)
🤖 - **Capitalization:** Full, 20% float (increases by phase)
🤖 - **Shares:** 20% president + 8x10%
🤖 - **Sell Movement:** down per 10% sold
🤖 - **Unsold Dividends:** market -> corp
🤖 - **Pool Limit:** 80%
🤖 - **Sell Order:** sell/buy
- **Currency:** yen

## Specialties
- Building permits (A/B/C/D) gate track laying per phase
🤖 - Communist takeover at phase B3 closes stock market
🤖 - Stock exchange reopens at phase C2
- Foreign investors
🤖 - Token cost doubled at phase D1
- 14 corporations
- Draft for 8 privates

## Stock Market Movement
🤖 - **Pay dividend:** right 1 (up if at edge)
🤖 - **Withhold:** left 1 (down if at edge)
🤖 - **Sell shares:** down 1 per 10% sold
🤖 - **Sold out:** up 1 at end of SR

## Auction
🤖 Draft -- each player picks one private.

## Round System
🤖 - SR -> OR x1 (phases A1--A2)
🤖 - SR -> OR x2 (phases B1--D3)
🤖 - Game ends after first 8-train triggers one more full OR set
