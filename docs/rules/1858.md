# 1858 — Rules Reminder

## Quick Reference
- **Players:** 2--6
- **Bank:** Pt8,000 (2p) / Pt12,000 (3p) / Pt12,000 (4p) / Pt12,000 (5p) / Pt12,000 (6p)
- **Starting Cash:** Pt500 (2p) / Pt500 (3p) / Pt375 (4p) / Pt300 (5p) / Pt250 (6p)
- **Capitalization:** Incremental, 40% float
- **Shares:** 20% president + 8x10%
- **Sell Movement:** down per share
- **Unsold Dividends:** market -> corp
- **Pool Limit:** 50%
- **Sell Order:** sell/buy
- **Half Pay:** Yes
- **Currency:** Pt (Peseta)

## Specialties
- Private railways can start public companies ;(
- Broad/narrow gauge track ;(
- Iberian Peninsula setting (Spain + Portugal) ;(
- H-train and M-train variants (4H/2M, 6H/3M, etc.) ;(

## Issue / Redeem
No issue/redeem. Incremental capitalization but private railways have separate mechanics.

## Loans
No loans.

## Stock Market Movement
- **Pay dividend:** right 1 (up if at edge)
- **Half pay:** no movement
- **Withhold:** left 1 (down if at edge)
- **Sell shares:** down 1 per share sold
- **Sold out:** up 1 at end of SR

## Auction
English auction -- ascending bids on each private.

## Round System
- SR -> OR x2 (all phases)
