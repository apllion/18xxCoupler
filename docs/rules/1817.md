# 1817 — Rules Reminder

## Quick Reference
- **Players:** 3--12
- **Bank:** Unlimited
- **Starting Cash:** $420 (3p) / $315 (4p) / $252 (5p) / $210 (6p) / $180 (7p) / $158 (8p) / $140 (9p) / $126 (10p) / $115 (11p) / $105 (12p)
🤖 - **Capitalization:** Incremental, 20% float
🤖 - **Shares:** 20% president + 1x10% (2-share corps; grows via conversion)
🤖 - **Sell Movement:** none (selling does NOT move price)
🤖 - **Unsold Dividends:** IPO -> corp
🤖 - **Pool Limit:** 1000% (effectively unlimited)
🤖 - **Sell Order:** sell/buy
🤖 - **Half Pay:** Yes
🤖 - **Loans:** Yes ($100 per loan, 5%+ escalating interest)
🤖 - **Shorts:** Yes (min 5-share corps; no new shorts in phase 8)
- **Currency:** $

## Specialties
🤖 - Corp sizing: 2-share -> 5-share -> 10-share via Merger & Conversion Round
🤖 - Seed money: $200 initial treasury per corp
🤖 - Train export: if no train bought in an OR, next available is removed
- Selection auction for 11 privates

## Issue / Redeem
🤖 Incremental cap with loans. Corps can take loans for emergency funds. No standard issue/redeem shares.

## Stock Market Movement
🤖 - **Pay dividend:** right 1
🤖 - **Half pay:** no movement
🤖 - **Withhold:** left 1
🤖 - **Sell shares:** NO movement
🤖 - **Sold out:** up 1 at end of SR

## Auction
🤖 Selection auction -- players bid on individual privates from a set of 11.

## Round System
🤖 - SR -> OR x2 -> Merger & Conversion Round (all phases)
🤖 - Train export at end of each OR if no train purchased
